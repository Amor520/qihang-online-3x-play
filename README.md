# 倍速播放脚本

这是一个给油猴（Tampermonkey）使用的定制脚本，目标站点是启航录播页面：

- `https://pre.iqihang.com/ark/record/*`

## 功能

- 长按键盘右方向键 `ArrowRight` 超过 `150ms`，临时切到 `3x`
- 松开右方向键后，恢复到页面原本的播放速度
- 短按右方向键时，尽量保留页面自带的快进行为
- 按 `F` 键：进入/退出播放器**原生全屏（不是网页全屏）**
- 全屏时空闲一小段时间后，自动隐藏鼠标指针；再次移动鼠标会恢复显示
- 新打开页面或刷新后，自动尝试开始播放当前视频
- 当前视频播放结束后，自动播放右侧章节列表里的下一节
- 如果上一节结束时仍在原生全屏，脚本会尽量直接保持全屏切到下一节
- 右侧“章节目录”面板默认自动收起
- 刷新或关闭标签页时，默认拦截站点自己的离开确认弹窗
- 放行 `Command + 1~9`、`Command + W`、`Command + R`，避免页面脚本拦截 Chrome 快捷键

## 当前文件

- `tampermonkey-bjy-right-click.user.js`

## 可调参数

脚本顶部可以直接改这几个常量：

- `TARGET_RATE = 3`
- `HOLD_DELAY = 150`
- `FORWARD_KEY = 'ArrowRight'`
- `FULLSCREEN_KEY = 'f'`
- `CURSOR_HIDE_IDLE_DELAY = 1400`
- `AUTO_PLAY_RETRY_DELAYS = [120, 500, 1200, 2400, 4000, 6500]`

## 快捷键说明

- `ArrowRight` 短按：尽量保持页面原本快进
- `ArrowRight` 长按：进入临时 `3x`
- `F`：进入/退出播放器原生全屏（不是网页全屏）
- 全屏时静止约 `1.4s`：自动隐藏鼠标；移动鼠标后恢复显示
- 页面打开或刷新后：自动尝试播放
- 当前视频结束时：自动点下一节
- `Command + R`、`Command + W`：不再弹出站点离开确认
- `Command + 1~9`、`Command + W`、`Command + R`：继续由 Chrome 处理

## 最近修复

- 去掉底部控制条自动隐藏逻辑，避免和播放器自身行为打架
- 保留原生全屏下的鼠标自动隐藏
- 下一节自动播放时，补上原生全屏的保留逻辑

## 安装方式

1. 在 Chrome 中安装 Tampermonkey
2. 打开篡改猴的“实用工具”
3. 将 `tampermonkey-bjy-right-click.user.js` 导入进去
4. 刷新启航录播页面测试

## 适用范围

这个脚本不是通用视频倍速脚本，而是偏站点定制脚本。

原因：

- 它只匹配 `pre.iqihang.com/ark/record/*`
- 它依赖启航播放器里的倍速控件类名，比如 `.ccH5sp`、`.ccH5spul`
- 它的目标是“保留站点原本右方向键快进逻辑的同时，额外加一个长按临时 3 倍速”

所以如果换网站、换播放器，或者启航页面改版，这个脚本都可能需要跟着调整。

## 备注

如果后面想做成更通用版本，通常要把这部分重构掉：

- 站点专用的选择器
- 站点专用的倍速菜单交互
- 只针对启航页面的 `@match`
