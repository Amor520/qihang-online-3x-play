# 倍速播放脚本

这是一个给油猴（Tampermonkey）使用的定制脚本，当前目标站点是启航录播页面：

- `https://pre.iqihang.com/ark/record/*`

## 功能

- 长按键盘右方向键 `ArrowRight` 超过 `150ms`，临时切到 `3x`
- 松开右方向键后，恢复到长按前的真实播放速度
- 短按右方向键时，尽量保留页面自带的快进行为；播放器原生全屏下会改为快进 `5s`
- `Y/H/G/J` 分别映射到 `上/下/左/右` 四个方向键
- 按 `L` 键时，如果当前实际倍速不是 `3x`，就切到 `3x`；如果当前已经是 `3x`，就切回 `1x`
- 鼠标仍然可以直接在播放栏里手动切换 `1.25x`、`1.5x`、`2x` 等其他倍速，脚本不会自动改回去
- 按 `F` 键，或点击播放器自己的全屏按钮，进入/退出播放器原生全屏
- 全屏时空闲一小段时间后，自动隐藏鼠标；再次移动鼠标会恢复显示
- 页面打开或刷新后，自动尝试开始播放当前视频
- 当前视频播放结束后，自动播放右侧章节列表里的下一节
- 如果上一节结束时仍在原生全屏，脚本会尽量保持全屏切到下一节
- 右侧“章节目录”面板默认自动收起
- 刷新或关闭标签页时，默认拦截站点自己的离开确认弹窗
- 放行浏览器快捷键 `Command/Ctrl + 1~9`、`Command/Ctrl + W`、`Command/Ctrl + R`

## 快捷键说明

- `ArrowRight` 短按：尽量保持页面原本快进；全屏时改为快进 `5s`
- `ArrowRight` 长按：进入临时 `3x`
- `Y/H/G/J`：分别等价于 `ArrowUp` / `ArrowDown` / `ArrowLeft` / `ArrowRight`
- `L`：在 `3x` 和 `1x` 之间快速切换
- 手动鼠标切到 `2x` 后，再按一次 `L` 会切到 `3x`，再按一次会切回 `1x`
- `F`：进入/退出播放器原生全屏
- 全屏时静止约 `1.4s`：自动隐藏鼠标；移动鼠标后恢复显示

## 当前文件

- `tampermonkey-bjy-right-click.user.js`

## 可调参数

脚本顶部可以直接改这几个常量：

- `TARGET_RATE = 3`
- `NORMAL_RATE = 1`
- `HOLD_DELAY = 150`
- `FORWARD_KEY = 'ArrowRight'`
- `FULLSCREEN_KEY = 'f'`
- `FIXED_RATE_TOGGLE_KEY = 'l'`
- `FULLSCREEN_FORWARD_SEEK_SECONDS = 5`
- `CURSOR_HIDE_IDLE_DELAY = 1400`
- `AUTO_PLAY_RETRY_DELAYS = [120, 500, 1200, 2400, 4000, 6500]`

## 最近更新

- `L` 的语义调整为“快捷切到 `3x/1x`”，不再锁死播放器倍速
- 播放栏手动切换 `2x`、`1.5x` 等其他倍速现在会正常保留
- 修复了“先手动切到 `2x`，再按两次 `L` 不能回到 `1x`”的问题

## 安装方式

1. 在 Chrome 中安装 Tampermonkey
2. 打开 Tampermonkey 的“实用工具”
3. 将 `tampermonkey-bjy-right-click.user.js` 导入进去
4. 刷新启航录播页面测试

## 适用范围

这个脚本不是通用视频倍速脚本，而是偏站点定制脚本。

原因：

- 它只匹配 `pre.iqihang.com/ark/record/*`
- 它依赖启航播放器里的倍速控件类名，比如 `.ccH5sp`、`.ccH5spul`
- 它的目标是“保留站点原本右方向键快进逻辑的同时，额外加一个长按临时 `3x`”

所以如果换网站、换播放器，或者启航页面改版，这个脚本都可能需要跟着调整。

## 备注

如果后面想做成更通用版本，通常要把这部分重构掉：

- 站点专用的选择器
- 站点专用的倍速菜单交互
- 只针对启航页面的 `@match`
